# PokemonApp
iOS application built with Swift and PokeApi

How to run app:
1. Clone this repository
2. Pod install on project folder
3. Open PokemonApp.xcworkspace

Tech Stack:
- Swift with Storyboard
- Pokeapi
- Firebase Authentication
- Firebase Realtime Database
